# 中文Blender手册报错(Simplified Chinese Blender Manual issue tracker)
## 报错站转移到GitHub的原因
1. 原本的手册报错点在百度贴吧，但是由于贴吧经常会无缘无故删贴，一些用户回复都被吞掉了。而GitHub则不会有百度贴吧那些破事发生。

2. GitHub的 **Issue** 是一种更加合理的报错途径，用户可以对**每个手册页面**或者**同类错误**开一个issue报错，而不用像贴吧那样无限码楼。
